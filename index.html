Course Data Architecture Blueprint

Create a directory using lowercase alphanumeric characters and hyphens only (e.g., my-custom-course). Inside this directory, place exactly four files matching the blueprints below:

1. config.json (Identity Metadata)

{
  "id": "my-custom-course",
  "title": "Clear Title (e.g., AWS Cloud Practitioner)",
  "description": "A single-sentence course summary.",
  "author": "Your Full Name",
  "version": "1.0.0"
}

2. tests.json (Exam Payload)

Grading maps to a zero-indexed integer (answer_idx: 0 to 3). 

{
  "test_1": [
    {
      "id": "q_unique_prefix_001",
      "question": "Question text without option letters.",
      "options": [
        "Choice 0",
        "Choice 1",
        "Choice 2",
        "Choice 3"
      ],
      "answer_idx": 1,
      "explanation": "Detail why Choice 1 is correct."
    }
  ]
}

3. flashcards.json (Active Recall)

[
  {
    "id": "fc_unique_prefix_001",
    "front": "Question visible on card front.",
    "back": "Answer revealed when flipped."
  }
]

4. notes.json (Study Material)

{
  "chapters": [
    {
      "chapter_idx": 1,
      "title": "Chapter One Title",
      "sections": [
        {
          "heading": "Section Subheading Name",
          "body": "Raw paragraphs of educational text."
        }
      ]
    }
  ]
}

Submit to GitHub

To open a Pull Request, we need a GitHub Personal Access Token with repo write scopes. Generate Token

🔒 Critical Step: How to Generate Your GitHub Access Token

Your token remains volatile in client RAM and is never stored. It is used exclusively to dispatch the Pull Request. Follow these exact parameters:

  1. Click the "Generate Token" configuration redirect link above.
  2. Authenticate your identity context with your personal GitHub account.
  3. In the Note configuration field, enter: KUMT-Course-Contribution.
  4. Set the Expiration boundary to 7 Days to maintain high security hygiene.
  5. Under Select Scopes, check the box for public_repo (or repo for private organizational structures). Leave all other checkboxes completely blank.
  6. Scroll to the absolute lower boundary of the portal and click the green "Generate token" action button.
  7. Copy the output string token (prefixed with ghp_) and paste it directly into our interface field above.
